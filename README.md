# FitOn

상황별 AI 코디 추천 쇼핑 서비스입니다. 사용자가 상황을 선택하거나 자연어로 검색하면 네이버 쇼핑 API 결과를 카드 형태로 보여줍니다.

## 주요 기능

- 상황별 코디 추천: 수련회, 롯데월드, 결혼식, 체육대회, MT, 반티, 일상, 데이트 상황에 맞는 상품을 검색합니다.
- 자연어 검색: 사용자가 입력한 문장을 검색 키워드로 변환해 원하는 분위기의 코디를 찾습니다.
- 스타일 조건 선택: 베이직, 스포티, 러블리, 스트릿, 포멀 스타일을 검색어에 함께 반영합니다.
- 예산 및 가격대 필터: 예산 입력, 빠른 예산 선택, 가격대 필터로 결과 범위를 좁힙니다.
- 네이버 쇼핑 연동: 서버가 네이버 쇼핑 API를 호출하고 상품명, 가격, 쇼핑몰, 이미지, 링크를 카드로 표시합니다.
- 상품 정렬 및 더보기: 관련도순, 최신순, 낮은가격순, 높은가격순 정렬과 추가 결과 로딩을 지원합니다.
- 보유 의류 등록: 사용자가 가진 옷을 브라우저 로컬 저장소에 저장하고 코디 조합에 우선 반영합니다.
- 추천 코디 조합: 추천 상품과 보유 의류를 묶어 추가 구매 비용, 예산 적합 여부, 할인 정보를 함께 보여줍니다.

## 실행 방법

```bash
npm install
copy .env.example .env
npm start
```

브라우저에서 `http://localhost:3000`에 접속합니다.

## 환경 변수

`.env`에 네이버 개발자 센터에서 발급받은 값을 넣습니다.

```env
NAVER_CLIENT_ID=발급받은_클라이언트_ID
NAVER_CLIENT_SECRET=발급받은_클라이언트_SECRET
```

포트를 바꾸려면 `.env`에 `PORT=3001`처럼 추가합니다.

## 주의

`.env`와 `node_modules/`는 Git에 올리지 않습니다. 의존성은 `package-lock.json` 기준으로 `npm install`로 복원합니다.
